# Amazon Review Sentiment Analysis using VADER & RoBERTa

## Overview

An NLP-powered sentiment analysis project that evaluates customer opinions from Amazon product reviews using both traditional and transformer-based approaches. The project compares VADER's lexicon-based sentiment scoring with RoBERTa's contextual language understanding to identify Positive, Negative, and Neutral sentiments, providing valuable insights into customer feedback and product perception.

## Key Highlights

- Performed sentiment analysis on Amazon customer reviews
- Implemented VADER and RoBERTa models for sentiment classification
- Compared rule-based and transformer-based approaches
- Conducted exploratory data analysis and sentiment visualization
- Generated customer insights from large-scale review data

## Technologies Used

- Python
- Pandas
- NumPy
- NLTK (VADER)
- Hugging Face Transformers
- RoBERTa
- Matplotlib
- Seaborn
- Jupyter Notebook

## Results

The comparative analysis demonstrated that RoBERTa delivers more accurate sentiment predictions by understanding contextual meaning, while VADER provides a lightweight and computationally efficient solution for rapid sentiment analysis.
